## 📜 Overview

The **Holographic Epigraphic Matrix** is an immersive 3D web application that visualizes six of the most critical early Islamic inscriptions. Each inscription is represented as a holographic marker on an interactive map, allowing users to explore epigraphic data, Arabic/English translations, spectrometry metrics, geographical significance, and credible APA references in a futuristic academic interface.

🔗 **Live Demo:** [https://yourusername.github.io/Holographic-Epigraphic-Matrix/](https://yourusername.github.io/Holographic-Epigraphic-Matrix/)

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🗺️ **Interactive 3D Map** | Explore 6 inscriptions on a holographic globe with drag, zoom, and rotate controls |
| 📜 **Epigraphic Data** | Access Arabic text, English translations, and historical context |
| 🔬 **Spectrometry Metrics** | View lab analysis data including substrate formation and elemental profiles |
| 📚 **APA References** | Credible academic sources with links to publications and UNESCO listings |
| 🔊 **Audio Narration** | Listen to inscription names spoken aloud with one click |
| 🖱️ **3D Rock Models** | Stylized 3D representations with photographic textures |

## 🏛️ The 6 Inscriptions

| # | Inscription | Date | Location | Significance |
|---|-------------|------|----------|--------------|
| 1 | **Zuhayr Inscription** | 24 AH | AlUla, Saudi Arabia | UNESCO Memory of the World |
| 2 | **Ghar al-Khail Cave** | 24 AH | Mada'in Salih, Saudi Arabia | Saudi Heritage Commission |
| 3 | **Mu'awiya Dam** | 58 AH | Wadi al-Khanaq, Ta'if | Umayyad civil engineering record |
| 4 | **Throne Verse Graffito** | c. 60 AH | Al-Hawra Coastal Outpost | Early Quranic inscription |
| 5 | **Usays Graffito** | 93 AH | Jabal Usays, Syria | Umayyad military frontier |
| 6 | **Darb Zubayda Boundary** | c. 100 AH | Dhali' al-Nis, Hail | Pilgrim route inscription |
